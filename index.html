Intégration et personnalisation d'une maquette

Cette page est un projet d'intégration d'une maquette. C'est le premier projet que j'ai réalisé pendant ma formation. La conception est basée sur les principes de base du développement front-end, à savoir les languages HTML (mise en page), CSS (mise en forme) et Javascript (intéractivité).

ONE-page

Cette maquette représente un site "vitrine" qui permet de résumer rapidement une activité sur une page unique, limitant ainsi la navigation pour permettre de rendre l'information accessible rapidement.

Responsive

Le rendu de la page change en fonction de l'appareil utilisé : téléphone, tablette, ordinateur. La présence et la disposition de certaines fonctionnalité dépends de la taille de l'écran.

Simple

La construction de cette page a été réalisé sans frameworks ni librairies : uniquement des flexboxs et du Javascript dit "vanilla".

Images
