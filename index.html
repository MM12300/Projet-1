Intégration et personnalisation d'une maquette

Cette page est un projet d'intégration d'une maquette. C'est le premier projet que j'ai réalisé pendant ma formation. La conception est basée sur les principes de base du développement front-end, à savoir les languages HTML (mise en page), CSS (mise en forme) et Javascript (intéractivité).

1-page

Cette maquette représente un site "vitrine" qui permet de résumer rapidement une activité.

Responsive

Le rendu de la page change en fonction de l'appareil utilisé : téléphone, tablette, ordinateur. La présence et la disposition de certaines fonctionnalité dépends du matériel du lecteur.

Simple

La construction de cette page a été réalisé avec les technologies de base, c'est à dire sans framework ni librairies.

Images

Toutes les images sur cette page (et dans mes projets persos en général) sont issues de ma bibliothèque personnelle.

Responsive

Cette page est s'adapte à votre appareil : téléphone, tablette ou ordinateur
